# Tarea Despliegue Aplicaciones

Proyecto de ejemplo para demostrar el uso de GitHub Actions, Issues y Proyectos Kanban.

## Estructura del Proyecto

- `index.html` - Pagina principal
- `styles.css` - Estilos CSS
- `app.js` - JavaScript de la aplicacion
- `package.json` - Configuracion del proyecto
- `.github/workflows/deploy.yml` - Workflow de CI/CD

## GitHub Actions

Este proyecto incluye un workflow automatizado que:

1. Verifica la estructura del codigo
2. Ejecuta linting
3. Ejecuta tests
4. Valida archivos requeridos para despliegue
5. Simula el despliegue

## Como usar

1. Clonar el repositorio
2. Abrir `index.html` en el navegador
3. Los cambios en `master` activan el workflow automaticamente
